# Youyouaidi

Ruby Gem youyouaidi offers a UUID class for parsing, validating and encoding UUIDs

## Installation

Add this line to your application's Gemfile:

    gem 'youyouaidi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install youyouaidi
